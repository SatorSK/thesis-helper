# 졸업논문 작성 도우미 (Thesis Helper)

국제통상 졸업논문을 **동국대 국제통상학과 양식**에 맞춰 쓰도록 돕는 로컬 웹앱.

> ⚖️ **이 도구는 표절·AI탐지 "우회" 도구가 아닙니다.**
> 초안·자료·인용은 도구가, **모든 본문 문장은 본인이** 작성합니다.
> 표절률을 낮추는 정공법 = 정확한 인용 + 본인의 독창적 분석. 그걸 돕는 도구입니다.

## 무엇을 하나 (7개 모듈)

| 모듈 | 기능 |
|------|------|
| ① 주제 발굴 | 2026 국제통상 이슈 기반 참신한 주제 후보 + 데이터 확보 난이도 점수 |
| ② 자료·인용관리 | 출처 입력 → **각주(학과 규정) + APA 참고문헌** 자동 생성 |
| ③ 골격 빌더 | 혼합형 학부논문 표준 골격, 분량(20쪽) 게이지 |
| ④ 데이터 분석 | CSV 업로드 → 추세/비교 그래프 + "데이터 사실" 요약 |
| ⑤ 단락 작성소 | 초안은 **참고용(복붙 금지)**, 본인 작성분만 논문에 포함. 유사도 경고 |
| ⑥ 자가 점검 | 인용 누락·AI 일반론·복붙 위험을 **할 일 목록**으로 표시 |
| ⑦ 내보내기 | 학과 양식 .docx (여백 30mm·11pt·160%·각주·표지·파일명 자동) |

## 동국대 국제통상학과 규정 (반영됨)

- 용지 A4, 여백 상하좌우 30mm, 머리말·꼬리말 15mm
- 본문 11pt, 주석 9pt, 줄간격 160%
- 본문 20장 이상 (표지·목차·참고문헌 제외)
- **인용은 각주 필수**, 참고문헌은 목록으로
- 카피킬러 유사도 **30% 이하**, 결과보고서 첨부 제출 (itrade@dongguk.edu)
- 파일명 `(국제통상학과)학번_이름`

> ⚠️ 위 규정은 2024학년도 공지 기준. **최신 졸업논문 공지를 학과에 재확인**하세요.

## 설치 & 실행

```bash
pip install -r requirements.txt
streamlit run app.py
```

브라우저에서 자동으로 열립니다. 좌측 사이드바에서 프로젝트 생성 → 모듈 순서대로 진행.

## LLM 백엔드 (선택, provider-agnostic)

기본은 **템플릿 모드(키 불필요)** 로 동작합니다. 더 풍부한 초안을 원하면 사이드바에서 백엔드 선택:

| 백엔드 | 필요 |
|--------|------|
| `openai` | OpenAI/Codex API Key (Base URL 변경 가능) |
| `anthropic` | Anthropic API Key |
| `gemini` | Google API Key |
| `local` | Ollama·LM Studio 등 OpenAI 호환 로컬 서버 (키 불필요) |

> API 키는 **세션에만 보관**하며 디스크에 저장하지 않습니다. 논문 원고 전체를 외부로 전송하지 않고, 해당 단락의 논점·인용만 전송합니다.

## 프로젝트 구조

```

core/
  project.py        # project.json 상태 저장/불러오기
  llm.py            # provider-agnostic LLM 어댑터
  citation.py       # 각주 + APA 포맷터
  docx_footnotes.py # 실제 OOXML 각주 주입 엔진
modules/m1~m7.py    # 7개 모듈 (각자 render(project, cfg))
projects/<name>/    # 프로젝트별 project.json, figures/  (gitignore)
```
